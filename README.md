# Customer Churn Prediction

Machine learning project to predict telecom customer churn using EDA, preprocessing, Random Forest, threshold tuning, and model evaluation.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Final Results
- Accuracy: 72.7%
- Precision: 49.1%
- Recall: 83.7%
- F1 Score: 61.9%
- ROC-AUC: 84.1%
